Matthew Bohnsack's MS Thesis

Making Creative Commons More Useful with Usage Management

This thesis proposes and implements an architecture that can be used to make Creative Commons more useful, by applying usage management.

Authors

Matthew Bohnsack (bohnsack@gmail.com)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/bohnsack/Matthew_Bohnsack_MS_Thesis

You can also directly download (unfinished and under-development) PDF versions of the actual paper and the presentation used during the defense.

Related Projects

During discussions held in UNM ECE informatics group research meetings, we have come up with at least three follow-on project ideas related to this work that might be used to by undergraduate and graduate students as a basis for their own projects:

1. Website to view, add, and/or edit CC license information embedded with XMP in various media: images, etc.

2. Implementation and further exploration of the "CC+ Broker Service" concept

3. Mobile phone application that automatically embeds XMP CC license information into photos and video
